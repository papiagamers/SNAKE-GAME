**PapiaGamer Snake Game**

Papia Gamers Snake Game is a modern, interactive Snake game built with **Python 3** using the **Tkinter** GUI framework. This game combines classic Snake gameplay with modern features such as **AI-controlled movement**, manual controls using **Arrow or W, A, S, D keys**, **level selection**, **fullscreen mode**, and **score tracking**. Designed for both beginners and experienced Python developers, itâ€™s an ideal project for learning game logic, GUI programming, and deploying Python applications using Docker.

## Features

- **AI Mode**: The snake automatically moves toward the food using a simple AI algorithm, making it fun to watch and educational for AI beginners.  
- **Manual Control**: Players can take over the snake at any time using **Arrow keys** or **W, A, S, D keys**.  
- **Level Selection**: Choose between Easy, Medium, and Hard levels to adjust the snake's speed for different skill levels.  
- **Main Menu**: An interactive main menu lets players start the game, select a level, or exit. The menu includes hover effects and sound cues for a more engaging experience.  
- **Fullscreen Mode**: The game dynamically adjusts to the screen size for an immersive experience.  
- **Score Tracking**: Real-time score and level display in the game interface.  
- **Sound Cues**: Simulated click, eating, and game-over sounds using the system bell.  
- **Docker Ready**: Run the game in a Docker container to maintain a consistent Python 3 environment.

## Installation

### Running Locally with Python 3

1. Clone the repository:
```
git clone https://github.com/papiagamers/snake-game.git
```
```
cd snake-game
```
```
python3 game.py
```

**How to Play  Snake Game:**

1. Launch the game to open the main menu.
2. Click Play to start the game.
3. Select a difficulty level: Easy, Medium, or Hard.
4. Use Arrow keys or W, A, S, D to control the snake manually.
5. If AI mode is enabled, the snake automatically moves toward the food.
6. Score increases each time the snake eats food.
7. The game ends if the snake collides with walls or itself. After Game Over, it automatically returns to the main menu.
8. You can return to the menu anytime using the Back to Menu button.
